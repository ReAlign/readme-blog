# 安装

# 安装 virtualbox

virtualbox 安装十分简单，可以一直 next, 也可以根据提示简单配置，没什么可讲的

# 安装 centos7

  • 新建虚拟机

  • 内存大小（自定义）

  • 虚拟硬盘（自定义）这里因为安装新系统，默认即可（现在创建虚拟硬盘）

  • 虚拟硬盘文件类型（默认）（VDI （VirtualBox 磁盘映像））

  • 存储在物理硬盘上， 选择固定大小，避免移植到其它电脑，出错的概率

  • 文件位置和大小（自定义）

  • 设置 —— 挂载盘片

  • 启动

  • 选择 Install CentOS 7 （默认： Test this media & Install CentOS 7）

  • 选择安装过程中的语言 （English 默认即可）

  • INSTALLATION SUMMARY

    • INSTALLATION DESTINATION

      • 进去之后选择 done 即可

    • SOFTWARE SELECTION

      • 选择 GNOME Desktop 桌面系统
      • 选择 Development Tools 开发者工具（必须，不然不能安装Virtualbox 增强工具）

    • NETWORK & HOSTNAME

      • 开启以太网
      • configure -> General -> 勾选 Automatically connect to this network when it is avaliable 自动连接到网路

  • 开始安装...

    • 设置 ROOT PASSWORD (密码可以简单设置)

    • 创建用户 USER CREATION (密码必须包含大小写字母和数字)

    • 耐心等待

  • 安装成功后，Reboot

  • LICENSE INFORMATION （版本信息），进去之后，勾选 accept -> Done 即可

  • 登录

# 系统配置

  • 禁止自动息屏

    • Power -> Power-saving -> Blank screen -> Never

  • 禁止锁屏

    • Privacy -> Screen Lock -> off

  • 时间设置（自动识别时区）

    • Details -> Automatic Time Zone -> ON

  • 免密登录

    • Details -> Users -> Automatic Login -> ON

# 增强功能

增强功能，可以使得 centos 桌面自适应屏幕， 支持文件夹共享，粘贴板共享

  • virtualbox菜单栏 设备

    • 安装增强功能 -> Run

  • 安装成功之后卸载掉 桌面的 VBox_CAs_..*, 右键 -> Eject

  • 共享文件夹

    • virtualbox 菜单栏 设置 -> 共享文件夹

    • 共享文件夹，只有 root 用户才有权限访问

  • 共享粘贴板

    • virtualbox 菜单栏 设置 -> 共享粘贴板 -> 双向
    • 重启服务
